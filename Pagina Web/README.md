# HelloMalaria
This is the README file for team 5

Este proyecto forma parte del PBL de Ingeniería Biomédica en Mondragon Unibertsitatea. El objetivo es desarrollar un software de análisis de imagen en MATLAB para asistir en el diagnóstico de malaria a partir de frotis sanguíneos digitalizados.

## Objetivo
Desarrollar un sistema que permita detectar automáticamente parásitos de malaria en imágenes microscópicas, proporcionando herramientas de apoyo al diagnóstico para técnicos de laboratorio y profesionales clínicos.
